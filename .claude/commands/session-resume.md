# Resume Session

Restore a previously saved session checkpoint.

## Usage

```bash
# Resume specific session
/session-resume session-YYYYMMDD-HHMMSS

# Resume latest saved session
/session-resume latest

# List available sessions
/session-resume list
```

## What Gets Restored

- ✅ Task context and goals
- ✅ Important decisions
- ✅ File references
- ✅ Session state

---

Resume sessions to continue work exactly where you left off.
